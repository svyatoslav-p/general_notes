Добавление нового статуса

Перейти по пути Администрирование –> Статусы задач добавляем задачу. Затем ее необходимо включить (не забываем про фильтр где можно выставить индивидуально для каждой Роль+Трекер) для этого после создания переходим Администрирование –> Последовательность действий –> Статус-переходы и отмечаем чек боксы на добавленной задаче например как на скриншоте и сохраняем
