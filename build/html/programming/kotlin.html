Активация¶

  1. Переходим на страницу и скачиваем файл ja-netfilter-all.zip

  2. Распакуем в любой каталог, но путь недолжен содержать пробелы и кириллицу. Например /home/svyatoslav/system/jetbrains/ja-netfilter-all/

  3. Выполним генерацию конфигурации /home/svyatoslav/system/jetbrains/ja-netfilter-all/scripts/install.sh

  4. Выйдем из системы и снова залогинимся

  5. Скопируем обновленную необходимую конфигурацию (в нашем случае для Intellij Idea Ultimate) из каталога /home/svyatoslav/system/jetbrains/ja-netfilter-all/vmoptions/idea.vmoptions в каталог /home/svyatoslav/.config/JetBrains/IntelliJIdea2021.3/idea.vmoptions

  6. Скопировать из каталога /home/svyatoslav/.config/JetBrains/IntelliJIdea2021.3/idea.vmoptions в этот же каталог, но с именем idea64.vmoptions

  7. Открываем файл /home/svyatoslav/system/jetbrains/ja-netfilter-all/config-jetbrains/mymap.conf устанавливаем любой срок действия лицензии в поле EQUAL,paidUpTo->

  8. Снова открываем страницу и копируем ключь продукта (кликнуть на нужной карточке)

  9. Открываем Intellij Idea Ultimate и вставляем ключ в поле Active Code
